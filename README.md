# RESQ OMAN

**RESQ OMAN** is an AI-assisted emergency command center prototype designed by **Salma Al Amri**. The project presents a smart concept for supporting search and rescue operations through a modern dashboard that helps monitor incidents, coordinate rescue teams, and simulate AI-powered decision support.

This prototype was developed as an academic project to demonstrate the potential use of Artificial Intelligence in emergency management. It uses simulated data to showcase the system's functionality and user experience.

## Features

* AI-assisted emergency response simulation
* Search and rescue command dashboard
* Incident monitoring interface
* Interactive charts and statistics
* Multi-role demo login
* Arabic and English language support
* Responsive and modern user interface

## Technologies

* HTML5
* CSS3
* JavaScript
* Chart.js
* Python (FastAPI)
